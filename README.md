# Titanic EDA Project

## What is this?
This is my first Exploratory Data Analysis (EDA) project using the Titanic dataset.

## What I learned
- How to load and inspect data using Pandas
- How to find missing values
- How to visualize data using Seaborn and Matplotlib
- How to find patterns in data

## Tools Used
- Python 3.11
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Jupyter Notebook in VS Code

## Key Findings
- Women survived more than men
- Richer passengers (1st class) survived more
- Children had higher survival rate
- Being adult male was the biggest risk factor
